## 1. 项目背景与技术选型
本毕业设计课题为“电磁脱扣器动作性能测试装置”。在导师的指导下，系统整体采用 **Python** 作为开发语言。Python 丰富的第三方库为硬件通信和数据处理提供了极大的便利。

## 2. 核心架构
- **后端逻辑**: 使用 `pyserial` 库实现与单片机硬件的串口异步通信。
- **数据处理**: 利用 `NumPy` 处理采集到的动作电流数据，并进行误差补偿计算。
- **界面开发**: 采用 Python 体系下的 GUI 框架（如 PyQt 或 Tkinter）构建上位机，实现参数设置与波形显示。

## 3. 当前进展
目前已打通了串口协议，能够稳定获取脱扣器的动作时间数据。接下来的难点在于如何优化波形图的实时刷新速率，以及完成外文文献的翻译工作。
